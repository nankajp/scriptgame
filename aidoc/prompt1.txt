# あなたの役割
あなたは優秀なゲームプログラマです。
とくに、javascriptに長けています。
依頼主である私の気持ちに寄り添った言葉は不要で、エンジニアとしてふるまってください。

# やりたいこと
以下のようなゲームを javascriptで作ってほしい。

1. 格子状の線図がゲーム画面である

　＿　＿　＿●
｜＿｜＿｜＿｜
｜＿｜＿｜＿｜
｜＿｜＿｜＿｜
●

2. プレイヤーは、各点で、棒を時計回りに90度ずつ回転させることができる
3. ●から●の位置まで、線が一筆でつながる（電源が通過する）ようにできたらゲームクリアである
4. 線は、必ずしもきれいな格子状にはならず、虫食いのような形になることもあり得る。以下の通り

　＿　＿　＿●
｜＿　＿　　｜
　＿｜　　　｜
｜＿｜＿　＿｜
●


# 作成するファイルについて
## フォルダ構成
aidoc      本プロンプトやあなたの作業ディレクトリ
css        cssファイルの置く場所
js         jsファイルを置く場所
index.html ゲーム画面

## ファイル構成
index.html ゲーム画面
xxx.js     ゲーム画面上で動作させる javascriptの外出しファイル
xxx.css    上記同様のcssファイル

## 注意
htmlファイルに直接jsコードを書き込まないこと
jsファイル、cssファイルは複数作成してよい
